城市超市会员卡种类

城市超市各门店推出会员卡，您也可以在网上购物货到付款时使用。本卡带会员照片，实行一卡双制，既是预付费储值卡，又是会员积分卡。

会员卡分如下四个等级：

  1. 食友/VIP
  2. 食客/Silver VIP
  3. 食家/Golden VIP
  4. 老饕/Diamond VIP

我们会为会员准备大小两张卡，方便您和家人同时使用，两张卡对应同一帐户，所有信息（会员信息、帐户信息）均保持一致。

会员卡积分功能

一、积分规则

  1. 会员卡持卡人分为四个等级：食友、食客、食家、老饕。当持卡人升级积分达到一定额度即可升级至更高级别，并提供免费的会员卡升级更换服务。

  2. 会员卡和积分规则如下表所示：

    会员级别 积分倍率 升级 续级
    食友 /VIP 1 升级积分至12000 续级积分不低于2400
    食客 /Silver VIP 3 升级积分至72000 续级积分不低于36000
    食家 /Golden VIP 5 升级积分至240000 续级积分不低于120000
    老饕 /Diamond VIP 7 续级积分不低于336000
    • 累计积分：自获得会员资格之日起累计获得积分总和。
    • 升级/续级积分：会员在连续24个自然月内获得的积分。若升级或续级从零计算。
    • 可用积分：会员目前有效的可兑换积分。
    • 若未达到续级标准，则降为下一级别会员或取消会员资格，升级/续级积分归零。
  3. 每张会员卡可使用年限为2年，2年满后须更换会员卡，办理会员卡更新，原卡内的金额和积分继续有效。
  4. 会员卡内的积分不得转让他人，不可兑换现金
  5. 会员的可用积分可以在城市超市各门店及城市超市网上商城兑换现金消费券或礼品。

二、积分计算

  1. 积分=∑（商品实际售价x单品积分倍率 x会员积分倍率）。
  2. 服务性消费（包括配送服务费、包装服务费）、香烟等，不享受积分累计。
  3. 使用现金、银行卡、信用卡、城市超市会员卡等卡种进行购物消费方可获得积分，其它任何形式的礼券、现金消费卡种均不可获得消费积分。

三、积分生效

  1. 在超市各门店消费及网上购物结账前出示会员卡，即可获取相应积分。
  2. 积分将于超市各门店或网上消费完成3天后正式生效。

四、积分清零

在每个自然年末将上一年度未使用（兑换）的可用积分自动清零。

例如，2012年12月31日将2011年1月1日—2011年12月31日间所积但未使用（兑换）的可用积分清零。

五、积分调整

如有任何商品退、换货等情况发生，我们将根据退换货产生的金额增减对您的积分做相应调整，以符合您实际消费所对应的积分。退换货请您凭原始收银条办理。

六、积分查询

或自行在网上商城进行积分查询，我们将为您保留有效期内的可用积分和累计积分信息，以便您随时查询。

查询方式如下：

  1. 收银条上显示三个工作日前的可用积分。
  2. 拨打400-811-1796服务热线进行查询。
  3. 就近到城市超市门店查询。
  4. 登陆城市超市网上商城自行查询
    我的账户 -> 会员设置 -> 会员积分 中进行查询。

七、积分兑换

会员可以用积分兑换城市好礼，包括制定商品、城市超市现金抵用券、礼券、会员活动、服务等。使用积分兑换所得的礼品不提供退、换服务，商品本身有质量问题除外。积分兑换在城市超市各门店服务台进行或登录城市超市网上积分商城自行兑换。

会员卡充值

会员卡具有充值功能，充值功能开通后，您可在卡内无息预存现金，用于消费时扣除。

  1. 充值方式：您可在城市超市各门店服务台，以现金或银联卡为您的会员卡充值并开具发票；其它卡种（斯玛特卡、杉德卡、联华OK卡，德仕卡、索迪斯卡、大众卡、魏康卡）充值时均不提供任何发票。充值一旦成功，除申请取消会员资格外金额不可退回。
  2. 卡内金额：会员卡内金额无有效期，只可在消费时使用，不折现、不透支。会员卡内的金额采用实时交易，可多次消费。
  3. 使用范围：会员卡内所存金额可在城市超市各门店购物使用
  4. 开具发票：会员充值时会开具发票，凭卡消费时则不再提供发票。

会员须知

一、会员申请

  1. 会员申请时须提供有效证件，证件类型详见申请表。
  2. 会员申请时，需要填写会员申请表，如申请人不能提供完整信息，城市超市有权不予发卡。
  3. 会员确认申请后，用户可以立即通过申请获得小卡，进行充值和消费积分累计。
  4. 主卡将于15个工作日内制作完毕后放至申请人申领门店并及时通知客人前往领取。

二、会员卡密码修改

会员卡密码包括储值账户密码（初始密码均默认为正面卡号后6位）和会员网上登录密码（初始密码均默认为背面卡号后6位）。请及时修改会员卡初始密码，并牢记新密码，若因您未更改卡片初始密码，您将承担因此产生的一切后果，本公司不承担任何责任。储值帐户密码可到城市超市各门店服务台修改。

三、会员卡挂失和补办

如您发现您的会员卡丢失、被盗、被抢或有他人冒用或盗用您的会员卡及密码或任何其他未经合法授权之情形时，应立即以有效方式通知本公司，要求本公司暂停相关服务。同时，您理解本公司对您的请求采取行动需要合理期限，在此之前，本公司对已执行的指令及（或）所导致的您的损失不承担任何责任。

  1. 会员卡挂失：会员可致电会员服务热线400-811-1796，进行挂失；或者到城市超市各门店服务台，向客服人员报失。
  2. 会员卡补办：会员持有效身份证件到任一超市门店填写会员补办申请表并补交工本费，新卡将于15个工作日内制作完毕后放至申请人补办门店并及时通知客人前往领取。
  3. 补办新卡中保留原现金账户和积分账户中余额。

四、会员卡冻结和注销

当如下任何一种情况发生时，城市超市会员资格将被冻结：

  1. 您名下的城市超市会员卡，一年内未有任何交易，会员卡将被冻结。
  2. 对捡拾卡、可疑卡、挂失卡，城市超市将直接冻结。
  3. 若发现申请人提供虚假信息或不完整信息，会员卡将被冻结

顾客若重新启用，须持卡主本人有效身份证件至城市超市各门店，经核实无误后，即可恢复使用。

当如下任何一种情况发生时，城市超市会员资格将被取消：

  1. 对每年1月1日至12月31日期间消费累计积分低于2400分的食友VIP会员，由于未达续级积分，城市超市将取消其会员资格。
  2. 您自愿申请取消会员资格。

对于冻结或注销的VIP会员，城市超市恕不通过任何方式传送信息与服务资料。

如需取消会员资格，请出示会员本人有效身份证原件及复印件、原始充值发票，经相关工作人员审核并签订《城市超市会员卡注销确认单》后，城市超市将于30个工作日内将扣税费后剩余的金额依据您的充值方式退还于您（若您以现金充值，则退还现金；若您以银行卡充值，则将退款至您充值时所提供的银行账户内），会员卡内有效的消费积分将自动失效。

若您无法提供上述材料，城市超市有权不予办理相关退款事项。
